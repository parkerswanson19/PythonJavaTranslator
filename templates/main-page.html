Python to Java Translator

This application translates Python and Java code. It is able to convert many different aspects of the languages and accomplishes it very swiftly.

Tools/Frameworks used

On the backend we used Python, and on the frontend we used CSS, HTML, and JavaScript. We also used Bootstrap, Django, and Heroku.

Rap Analyzer

This application takes a given rap song and returns an analysis of the lyrics. The user learns the number of expletives used, the average grade level the song was written at, and much more.

Tools/Frameworks used

We used Python for the backend, and HTML and CSS for the frontend. For the lyrics, the Genius API was used. Postgresql was used as the database, storing all of the songs analyzed.
