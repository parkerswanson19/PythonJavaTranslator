Official documentation for supported features:

  • Variable declarations
    • Warning: After a variable is declared in Python, it can be re-assigned to a different variable type. Java doesn’t support this.
    • ints
    • doubles
    • Strings
      • Warning: Strings have to use f-string formatting in order for them to be able to be translated to Java
    • booleans
    • lists/arrays
      • Warning: In Python, having multiple types of variables/objects in a list is allowed, in Java, arrays don’t support this feature.
      • Warning: Only Python supports reverse indexing.
      • Warning: Python substringing allows for a step parameter, Java’s substring method lacks this feature.
    • tuples
      • Warning: Only Python supports tuple unpacking in for loops.
  • Comments
  • Print statements
  • Mathmatical operations
    • Warning: Dividing two integers in Python will yield a double, where in Java, you’ll get an integer.
  • If statements
  • For loops
    • Warning: Python supports using a for loop to iterate through a string character by character, Java foreach loops don’t support this.
  • While loops
    • Warning: Java doesn’t have while-else functionality.
